# ucc-electiva-iv
## Semaforo Doble
Consta de un circuito  que representa la funcionalidad de un semáforo doble.
### Componentes requeridos para la contrucción 
1. Placa de arduino uno
2. Breadboard small
3. Bombillos led (diez)
4. Resistencias (diez de 330 hm)
5. Pins (diez)
6. jumper (dos)
